## Why This Matters

- **Crawled or OpenAlex metadata is not training data.**
- We must enrich with **licenses + PMC IDs** before downloading full text.
- Only **licensed OA full text** is eligible for training-derived Q&A.

## Scripts Added

1. `scripts/one_shot_real_data_pipeline.py`
   - Collects OpenAlex OA candidates and optionally downloads Europe PMC full text.
   - Uses cursor paging, resume, retries, and debug logging.

2. `scripts/enrich_candidates_with_europe_pmc.py`
   - Looks up each candidate by DOI in Europe PMC.
   - Adds:
     - `pmcid`
     - `europe_pmc_license`
     - `europe_pmc_journal`
     - `europe_pmc_enriched=true`
   - Supports concurrency + retries.

3. `scripts/download_europe_pmc_fulltext.py`
   - Downloads full‑text XML for enriched rows with allowed licenses.

4. `scripts/one_shot_train_pipeline.py`
   - Runs: download -> convert -> partition -> backfill/clean -> repartition -> validate
   - Optional: `--enable-template-augmentation` adds enrich + variate steps
   - Optional flags: `--skip-train`, `--skip-generate`, `--download-print-progress`, `--download-request-timeout`
